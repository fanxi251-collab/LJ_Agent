# Live2D evaluation assets

This directory vendors the following local-only evaluation resources:

- `mao_pro/`: Niziiro Mao (PRO Version) runtime model from [Open-LLM-VTuber](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber/tree/main/live2d-models/mao_pro), originally created by Live2D Inc.
- `live2dcubismcore.min.js`: Live2D Cubism Core redistributable code from the [Open-LLM-VTuber Web build](https://github.com/Open-LLM-VTuber/Open-LLM-VTuber-Web/blob/build/libs/live2dcubismcore.min.js).

The model is included to evaluate the local digital-human integration. Review the bundled sample-data terms and the Cubism proprietary software agreement before any public or commercial deployment.
